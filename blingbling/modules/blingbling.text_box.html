new (args, text)
Create a t_box widget.

Parameters

  • args:
  • text: a table : { text = your_text } No need to set width or height, it depends on padding, font size and text length.

Usage:

myt_box=blingbling.text_box.new({text = "your text" })

Return value:

A t_box widget.
