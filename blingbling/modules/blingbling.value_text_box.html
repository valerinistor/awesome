set_values_text_color (vt_box, table)
Define the value text color depending on the limit given ( if value >= limit, we apply the color).

Parameters

  • vt_box: the value text box
  • table: table like { {"#rrggbbaa", limit 1}, {"#rrggbbaa", limit 2}}

Usage:

myvt_box:set_values_text_color(table) -->{ {"#rrggbbaa", limit 1}, {"#rrggbbaa", limit 2}} By default the color is default_text_color(another example: {{"#88aa00ff",0},{"#d4aa00ff", 0.5},{"#d45500ff",0.75}})
